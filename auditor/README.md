## What it does ?

This tar will audit keybibding , functions , duplicate mappings.

## Usage 

```sh
# 1. Go to your downloads folder
cd ~/.config/nvim/auditor/  # or wherever it is

# 2. Extract it
tar -xzf dustnvim-auditor-complete.tar.gz

# 3. Go into the folder
cd dustnvim_audit  # or just ls to see what folder was created

# 4. Make executable and run
chmod +x install.sh audit_dustnvim.py
./install.sh
```
